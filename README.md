# Udemy Clone Website

A simple Udemy-inspired website created using only HTML and CSS. This project was built to practice web design fundamentals, page structure, and responsive layouts.

## Features

* Clean and modern Udemy-style interface
* Navigation bar and search section
* Course cards and category layouts
* Responsive design using CSS
* Beginner-friendly project structure

## Technologies Used

* HTML5
* CSS3

## Purpose

This project was created as part of my web development learning journey to improve my understanding of HTML and CSS by recreating the design of a popular online learning platform.

## Future Improvements

* Add JavaScript for interactivity
* Implement search functionality
* Create individual course pages
* Improve responsiveness and animations
